## 📌 Project Overview
This project focuses on developing digital literacy skills essential for students in today’s digital world. It includes practical tasks such as creating an infographic, building professional online profiles, designing a quiz, writing formal emails, and understanding cybercrime.

The portfolio demonstrates the ability to use digital tools effectively, communicate professionally, and follow safe online practices.
