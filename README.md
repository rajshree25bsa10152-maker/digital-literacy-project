# digital-literacy-project
## 📄 Project Report

For this task, I used Canva to design my digital literacy infographic because it provides user-friendly templates, icons, and layout tools suitable for beginners. My design presents the concept of digital literacy in a clear and visually structured format. It includes key sections such as the definition of digital literacy, useful digital tools for students like Google Docs and GitHub, safe internet practices such as using strong passwords and avoiding suspicious links, and the importance of maintaining a professional online presence. I also added a section on email etiquette to highlight formal communication skills.

One thing I found difficult was organizing all the information into a single slide without making it look crowded. However, this process was also interesting because it improved my creativity and helped me understand how to present information in a concise and visually appealing way.
